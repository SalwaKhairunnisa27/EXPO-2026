# EXPO 20.0 Website

Website resmi untuk kegiatan **EXPO 20.0** yang diselenggarakan oleh
**Unit Kegiatan Mahasiswa Kelompok Peneliti Muda (KPM)**
Universitas Negeri Jakarta.

---

## 📌 Tentang EXPO 20.0

EXPO 20.0 merupakan rangkaian kegiatan besar yang bertujuan untuk
menggelorakan budaya ilmiah di kalangan mahasiswa Universitas Negeri Jakarta.

Pada tahun 2026, EXPO menghadirkan beberapa agenda utama, yaitu:

* **CASTLE** (Conference on Social, Science, Technology, Language, and Education Research)
* **SKILL** (Seminar Keilmiahan dan Hasil Penelitian)

Website ini berfungsi sebagai media informasi dan publikasi kegiatan EXPO.

---

## 🎯 Tujuan Website

* Menyampaikan informasi terkait EXPO 20.0
* Memperkenalkan program utama (CASTLE, SKILL, PKIM)
* Menjadi media promosi kegiatan ilmiah mahasiswa
* Mengarahkan pengguna untuk mengikuti kegiatan
